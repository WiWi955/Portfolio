Installation et Sécurisation d’un Serveur FTP / FTPS

Ce tutoriel explique comment installer un serveur FTP basique et comment le convertir en FTPS pour plus de sécurité.

1. Introduction

Ce guide vous explique comment installer et sécuriser un serveur FTP / FTPS sous Debian 12, en passant par la configuration de vsftpd, l’ajout de FTPS avec TLS/SSL, et la protection avec fail2ban.

2. Mettre à jour le système

Pour mettre à jour la liste des paquets disponibles et installer les dernières versions des paquets déjà installés :

sudo apt update
sudo apt upgrade -y

3. Configurer une adresse IP statique

Ouvrez le fichier de configuration des interfaces réseau pour édition :

sudo nano /etc/network/interfaces

3.1 Ma configuration :

Ajoutez ou modifiez la configuration suivante pour définir une adresse IP statique :

auto eth0
iface eth0 inet static
address 192.168.102.110
netmask 255.255.255.0
gateway 192.168.102.1
dns-nameservers 8.8.8.8 8.8.4.4

3.2 Redémarrez les services réseau pour appliquer la nouvelle configuration :

sudo systemctl restart networking

4. Installation d’un Serveur FTP

4.1 Installation de vsftpd

Exécutez la commande suivante pour installer vsftpd sur Debian :

sudo apt-get install vsftpd

4.2 Vérifier la version de vsftpd :

vsftpd -version

4.3 Lancer le service FTP :

sudo systemctl start vsftpd

4.4 Lancer le service FTP à chaque démarrage du serveur :

sudo systemctl enable vsftpd

4.5 Vérifiez l’état du service FTP :

sudo systemctl status vsftpd

4.6 Configuration de vsftpd

Ouvrez le fichier de configuration :

sudo nano /etc/vsftpd.conf

Redémarrez vsftpd pour appliquer les changements :

sudo systemctl restart vsftpd

4.7 Configuration du Pare-feu (UFW)

Installez UFW :

sudo apt install ufw

4.8 Ouvrez les ports nécessaires :

sudo ufw allow 21/tcp  # Port du FTP/FTPS
sudo ufw allow 22/tcp  # Port du SSH
sudo ufw allow 40000:50000/tcp  # Mode passif du FTP/FTPS

4.9 Activez et vérifiez que UFW est bien actif :

sudo ufw enable
sudo ufw status

4.10 Vérification du Fonctionnement

Testez la connexion avec un client FTP comme FileZilla.

Voir les transactions FTP en temps réel :

sudo tail -f /var/log/vsftpd.log

Attention : Ce serveur FTP fonctionne, mais il n’est pas sécurisé ! Tout transite en clair, donc si quelqu’un intercepte les données, il verra tout (y compris les mots de passe). Si vous voulez un serveur sécurisé avec FTPS (chiffrement TLS/SSL), passez à la partie suivante.
